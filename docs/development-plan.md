# 房地产开发测算平台 — 建设规划

## 架构定位

**一个模板 + 一个引擎 = 一整条测算流水线**

不是拆成独立模板。开发测算各模块强关联（改地价→IRR变），必须在一个引擎内串联计算。

```
kaifa.yaml
 ├─ 土地信息
 ├─ 产品信息 + 单方成本
 ├─ 三费费率
 ├─ 融资参数
 ├─ 销售参数
 ├─ 税金参数
 └─ 运营/周期参数
         ↓
kaifa.py (计算流水线)
 ├─ 1. 土地成本       (出让金+契税+配套费)
 ├─ 2. 建安成本       (各产品类型×单方)
 ├─ 3. 三费           (管理/营销/代建)
 ├─ 4. 销售计划       (月度去化→收入)
 ├─ 5. 融资计划       (开发贷/前融/还本付息)
 ├─ 6. 税金           (增值税+土增税+所得税)
 ├─ 7. 现金流         (逐月 IRR/NPV/回正)
 └─ 8. 盈利汇总       (净利润/ROE)
         ↓
JSON 结果 → Vue3 展示
```

---

## P1 — 最小可用版（2-3 周）

**目标**：完整走通一条测算流水线，出 IRR 和净利润。

### 输入（YAML 参数）

```
土地:   面积、亩单价、契税税率、配套费标准
产品:   类型×面积×售价×单方造价 (高层/洋房/商业/地下)
三费:   管理费率、营销费率、代建费率 (可选)
融资:   开发贷额度/利率/期限、自有资金比例
销售:   去化周期、月度去化比例、回款比例
税金:   增值税预征率、土增税预征率、所得税率
周期:   拿地→开工→开盘→竣备→清盘 节点月数
```

### 输出

| 指标 | 来源 |
|---|---|
| 总投资额、单方成本 | 土地+建安+三费+资本化利息 |
| 总收入、总回款 | 销售计划 |
| 税前利润、净利润 | 收入-成本-税金 |
| IRR、NPV | 现金流 numpy-financial |
| 资金峰值、回正周期 | 现金流 |
| 利润率、ROE | 汇总 |

### 不做的

| 功能 | 原因 | 计划 |
|---|---|---|
| 月度现金流 | P1 做年度，够用 | P2 改月度 |
| 15种产品明细 | 压缩到4类(高层/洋房/商业/地下) | P2 扩充 |
| 土增税四级累进 | 先简化按预征率 | P2 加精确计算 |
| 前端融资/非标融资 | 只做开发贷 | P2 加 |
| 多地块分摊 | 先做单地块 | P3 后 |
| 监管资金 | 不影响 IRR 计算 | P3 后 |
| 敏感性分析 | 手动改参数对比 | P3 |

---

## P2 — 精度提升（1-2 周）

- 年度 → 月度现金流
- 产品类型从4类扩展到 8-10 类
- 土增税从预征率改为四级超率累进
- 增加前端融资 / 非标融资
- 增加资本化利息自动计算

---

## P3 — 决策支持（1-2 周）

- 敏感性分析（售价/地价/利率 ±10%、±20%，批量跑）
- 多方案保存与对比
- 导出 PDF / Excel
- 监管资金计算

---

## P4 — 高级功能（按需）

- 多地块分摊（土地费自动按面积分摊到各产品）
- 城市参数库 (data/cities.yaml，轻量级，不做云数据库)
- 跟投/激励机制

---

## 不做

- 绿城标准工期表（你们用自己工期）
- 代建费体系（不涉及代建模式）
- 标准配置库（每项目差异大，直接填单方）
- 一键导出无宏 Excel（Web 原生支持导出，不需要这个功能）
